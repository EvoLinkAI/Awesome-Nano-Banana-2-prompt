> Una colección curada de **más de 100 prompts de alta calidad** para Nano Banana 2 — desde retratos fotorrealistas hasta experimentos creativos, dioramas 3D y más.

Nano Banana 2 está construido sobre Gemini 3.1 Flash Image — ofreciendo calidad Pro a velocidad Flash. Este repositorio recopila los prompts más efectivos de creadores de X (Twitter) e ingenieros de prompts para ayudarte a desbloquear todo su potencial.
